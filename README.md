#  Project: **Simple FAQ Chatbot**
 **Goal**: Build a chatbot that answers predefined FAQ questions using structured responses.  
 **Tech Stack**: Python, OpenAI, Flask (for API), Streamlit (UI), JSON (to store FAQs).
